# A Vendor-Agnostic Framework for SOAR Evaluation Based on Common Attack Patterns

Supplementary material for a Master's thesis in Cyber Security at IU International University.

## Overview

This repository contains supplementary material for the Master's thesis:

“A Vendor-Agnostic Framework for SOAR Evaluation Based on Common Attack Patterns.”

The thesis follows a Design Science Research (DSR) methodology and proposes a framework for evaluating Security Orchestration, Automation, and Response (SOAR) platforms in a vendor-agnostic manner. The framework leverages common attack patterns to assess SOAR capabilities in a structured and reproducible way.

The materials in this repository support the thesis by providing artifacts, datasets, and documentation used during the design and evaluation of the proposed framework.

## Repository Contents

The repository contains the following types of materials:

- Framework artifacts  
  Components of the proposed SOAR evaluation framework.

- Attack pattern definitions  
  Structured representations of common attack scenarios used for evaluation.

- Evaluation materials  
  Supporting documents and data used during the framework validation.

- Supplementary documentation  
  Additional explanations and resources referenced in the thesis.

## Purpose of This Repository

This repository is intended to:

- Provide transparency for the research artifacts used in the thesis
- Enable reproducibility of the evaluation approach
- Serve as a reference implementation of the proposed framework
- Support further research and experimentation in SOAR evaluation

## Usage

The materials in this repository can be used to:

- Explore the structure of the proposed SOAR evaluation framework
- Review the attack patterns used in the evaluation process
- Reuse or adapt the artifacts for research or experimentation

Refer to the individual folders for detailed documentation.
